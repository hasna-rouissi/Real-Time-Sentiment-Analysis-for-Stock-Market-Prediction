# Real-Time Stock Market Sentiment Analysis

## Lancement

1. Installe Docker Desktop
2. Place-toi dans ce dossier :
   ```bash
   docker-compose up
   ```
3. Ouvre Jupyter Notebook : http://localhost:8888
4. Lance `kafka_producer.py` pour envoyer les tweets
5. Ouvre le notebook dans `notebooks/sentiment-analysis.ipynb`
